# Microsoft-Elevate-Internship-PowerBI-Project
This end-to-end Power BI based business intelligence solution was developed as part of the Microsoft Elevate Internship program.

# 🎯 Key Business Questions Answered
The dashboard is engineered to address critical enterprise requirements through interactive data storytelling:

Dynamic Performance Ranking: Utilized Field Parameters to allow users to instantly toggle Top 5 / Bottom 5 product rankings by Sales, Profit, or Quantity Sold.

Time Intelligence & Trend Analysis: Implemented a custom Date Table to track sales trajectories (Annually to Daily) and execute accurate Year-over-Year (YoY) comparisons.

Geographical Revenue Mapping: Visualized sales and profit concentration across global cities using interactive map visuals.

Profitability Correlation: Employed scatter plot analysis to visualize the direct relationship between total sales and profit per product, identifying high-revenue but low-margin outliers.

Granular Order Tracking: Engineered a comprehensive data matrix, cross-filtered by global slicers (Date, Product Line, Customer Segment, Promotion Category) for row-level auditing.
